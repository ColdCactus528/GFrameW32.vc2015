Проект GFrameW32 предназначен для визуализации результатов лабораторных работ по курсу "Основы компьютерной графики"

Работаем с файлом Graphics.cpp. В собственном коде предпочтительно использовать только функцию gfSetPixel для вывода пикселя на экран, 
функции gfGetWindowWidth и gfGetWindowHeight для определения размеров окна для вывода, а также функцию gfClearScreen для очистки экрана.

Вызов разработанного кода, выполняющего задания первой и второй лабораторных работ, следует поместить в gfInitScene. 
Вызов разработанного кода, выполняющего задания третьей лабораторной работы, следует поместить в gfDrawScene.
