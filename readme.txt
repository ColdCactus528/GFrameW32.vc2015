Проект GFrameW32 предназначен для визуализации результатов лабораторных работ по курсу "Основы компьютерной графики"

Непосредственная работа производилась с файлом Graphics.cpp. В данном репозитории находится лабораторная работа по визуализации параллелепипеда с разных сторон. Для запуска используйте GFrameW32.vc2015.sln.
